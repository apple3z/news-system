## 🔧 技术栈

### 前端
- Vue 3 + Vue Router 4
- 响应式 CSS 布局
- CSS 变量（暗色主题）
- localStorage 本地存储

### 后端
- Flask 框架
- SQLite 数据库（news.db, skills.db, subscribe.db）
- feedparser（RSS 解析）
- requests（HTTP 请求）
- BeautifulSoup4（HTML 解析）

### 核心算法
```python
# 权重评分算法
排序分数 = 源权重 × 时间衰减系数
时间衰减系数 = exp(-小时数 / 24)

# 示例：
# 源权重 = 5（5 星）
# 发布时间 = 12 小时前
# 时间衰减 = exp(-12/24) = exp(-0.5) = 0.606
# 排序分 = 5 × 0.606 = 3.03
```
