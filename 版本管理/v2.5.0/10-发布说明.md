## 六、回滚方案

### 6.1 回滚条件

- 核心功能不可用
- 数据丢失或损坏
- 严重性能问题

### 6.2 回滚步骤

```bash
# 1. 停止服务
pkill -f "python app.py"

# 2. 恢复备份
cp app.py.backup.v24 app.py

# 3. 删除新增文件
rm -rf utils/
rm static/sys_v2.js

# 4. 重启服务
python app.py &
```
