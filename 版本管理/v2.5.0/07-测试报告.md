## 三、详细测试结果

### 3.1 API测试结果

#### API-001: GET /api/v2/versions

**测试内容**: 获取版本列表

**测试步骤**:
```bash
curl http://192.168.36.133:5000/api/v2/versions
```

**期望结果**:
- 返回code: 200
- 版本按倒序排列（v2.5.0 > v2.4.0）
- 每个版本包含12个文档

**实际结果**:
```json
{
  "code": 200,
  "data": [
    {"version": "v2.5.0", "docs": 12个},
    {"version": "v2.4.0", "docs": 12个}
  ]
}
```

**结论**: ✅ **通过**

---

#### API-002: GET /api/v2/wiki/read

**测试内容**: 读取文档内容和版本信息

**测试步骤**:
```bash
curl http://192.168.36.133:5000/api/v2/wiki/read/版本管理/v2.5.0/02-需求.md
```

**期望结果**:
- 返回code: 200
- 包含content、version、mtime、locked字段

**实际结果**:
```json

  "code": 200,
  "version": "1.0.0",
  "mtime": "2026-03-06 08:07:41",
  "locked": false,
  "content": "# 产品需求文档..."
}
```

**结论**: ✅ **通过**

#### API-003: POST /api/v2/wiki/save

**测试内容**: 保存文档并更新版本

**测试步骤**:
```bash
curl -X POST http://192.168.36.133:5000/api/v2/wiki/save \
  -H "Content-Type: application/json" \
  -d '{"filepath": "版本管理/v2.5.0/07-测试报告.md", "content": "# 测试", "locked": false}'
```

**期望结果**:
- 返回code: 200

**实际结果**:
