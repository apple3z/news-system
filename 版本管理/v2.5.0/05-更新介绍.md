### 3.2 修复详情

#### B01 - 版本历史重复显示
**问题现象**: 页面显示两个版本历史区域

**根因分析**: 
- 代码同时从 versions.json 和目录扫描读取版本
- 两个数据源合并时未去重

**修复方案**:
```python
# 删除 versions.json 相关代码
# 只从 /版本管理/ 目录扫描
versions = scan_version_dirs('版本管理')
```

#### B02 - 文档点击无响应
**问题现象**: 点击版本目录下的文档没有反应

**根因分析**:
- 混合使用 `/doc/` 和 `/版本管理/` 路径
- 某些文档路径解析错误

**修复方案**:
```python
# 统一使用 /版本管理/ 路径
VERSION_DOCS_DIR = "/home/zhang/.copaw/news_system/版本管理"
```

#### B03 - 版本顺序错误
**问题现象**: 版本按 v2.4, v2.5, v2.6 正序排列

**根因分析**:
- 使用字符串排序而非版本号排序
- "v2.10" < "v2.5" （字符串比较）

**修复方案**:
```python
# 解析版本号为数字元组再比较
def parse_version(v):
    return tuple(int(x) for x in v.lstrip('v').split('.'))

versions.sort(key=parse_version, reverse=True)
```
