### ✅ B01 - 版本历史显示重复

| 属性 | 内容 |
|------|------|
| **编号** | B01 |
| **标题** | 版本历史显示重复 |
| **严重程度** | 🔴 P0 |
| **状态** | ✅ 已修复 |
| **修复时间** | 2026-03-06 16:30 |
| **修复人** | 瑞贝卡 |

**问题描述**: 系统管理页面中，版本历史区域显示两次。

**根因分析**: 代码同时从versions.json和目录扫描读取版本数据。

**修复方案**: 
- 新增sys_v2.js，使用versionsLoaded标志防止重复加载
- 只从/版本管理/目录扫描，删除versions.json依赖

**验证结果**: 
- API测试：✅ 通过（返回唯一版本列表）
- Web测试：⬜ 待执行
