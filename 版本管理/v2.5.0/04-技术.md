### 2.2 API接口实现

**文件**: `app.py`

#### V2 API路由

| 路由 | 方法 | 说明 |
|------|------|------|
| `/api/v2/versions` | GET | 获取版本列表（倒序） |
| `/api/v2/wiki/read/<path>` | GET | 读取文档内容和版本信息 |
| `/api/v2/wiki/save` | POST | 保存文档并更新版本 |

#### 关键代码

```python
@app.route('/api/v2/versions')
def api_v2_versions():
    """获取版本列表（倒序）"""
    versions = get_sorted_versions('版本管理')
    data = [{'version': v, 'path': f'版本管理/{v}'} for v in versions]
    return jsonify({'code': 200, 'data': data})

@app.route('/api/v2/wiki/read/<path:filepath>')
def api_v2_wiki_read(filepath):
    """读取文档"""
    # 安全验证
    if not validate_doc_path(filepath):
        return jsonify({'code': 400, 'msg': '无效路径'}), 400
    
    # 读取文件
    full_path = os.path.join(BASE_DIR, filepath)
    with open(full_path, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # 获取版本信息
    version_info = get_doc_version(filepath, BASE_DIR)
    
    return jsonify({
        'code': 200,
        'content': content,
        'version': version_info['version'],
        'mtime': version_info['mtime'],
        'locked': version_info['locked']
    })

@app.route('/api/v2/wiki/save', methods=['POST'])
def api_v2_wiki_save():
    """保存文档"""
    data = request.get_json()
    filepath = data.get('filepath')
    content = data.get('content')
    locked = data.get('locked')
    
    # 安全验证
    if not validate_doc_path(filepath):
        return jsonify({'code': 400, 'msg': '无效路径'}), 400
    
    # 保存并更新版本
    result = save_doc_with_version(filepath, content, BASE_DIR, locked)
    return jsonify(result)
```

## 三、Bug修复实现

### B01: 版本历史重复显示

**根因**: 同时从 versions.json 和目录扫描读取，且前端重复渲染

**修复**:
1. 后端只从目录扫描（删除 versions.json 依赖）
2. 前端添加 `versionsLoaded` 标志防重复

```javascript
let versionsLoaded = false;
async function loadVersionTree() {
    if (versionsLoaded) return;  // ✅ 防重复
    // ...
    versionsLoaded = true;
}
```

### B02: 点击文档无响应

**根因**: 混合使用 `/doc/` 和 `/版本管理/` 路径

**修复**: 统一使用 `/版本管理/` 路径

```python
# 安全验证
allowed_prefixes = ['版本管理/', 'docs/', '研发规范/']

```

### B03: 版本顺序混乱

**根因**: 字符串排序导致 v2.10 < v2.5

**修复**: 语义化版本号排序

```python
def parse_version(version_str):
    clean = version_str.lstrip('v')
    parts = clean.split('.')
    return tuple(int(p) for p in parts[:3])

def compare_versions(v1, v2):
    p1 = parse_version(v1)
    p2 = parse_version(v2)
    for a, b in zip(p1, p2):
        if a != b:
            return a - b
    return len(p1) - len(p2)

# 使用：sorted(versions, key=cmp_to_key(compare_versions), reverse=True)
```

## 七、安全设计

### 7.1 路径安全

```python
def validate_doc_path(filepath):
    # 禁止路径遍历
    if '..' in filepath:
        return False
    
    # 禁止绝对路径
    if filepath.startswith('/'):
        return False
    
    # 只允许特定目录
    allowed = ['版本管理/', 'docs/', '研发规范/']
    return any(filepath.startswith(p) for p in allowed)
```

ALLOWED_EXTENSIONS = {'.md', '.txt', '.json'}
