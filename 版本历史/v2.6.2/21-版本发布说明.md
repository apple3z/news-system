## 🔧 技术架构升级

### 核心算法

#### 权重评分算法
```python
# 排序分数 = 源权重 × 时间衰减系数
# 时间衰减系数 = exp(-小时数 / 24)

# 示例：
源权重 = 5 (5星)
发布时间 = 12小时前
时间衰减 = exp(-12/24) = exp(-0.5) = 0.606
排序分 = 5 × 0.606 = 3.03
```

### API 接口清单

#### Skills API（4个）
- `GET /api/skills/search` - 搜索 Skills（13个字段）

- `GET /api/skills/<id>` - 获取单个 Skill 详情
- `GET /api/skills/rankings` - 获取排行榜

#### 订阅 API（9个）
- `GET /api/subscribe/feed` - 订阅内容列表（增强版）
- `GET /api/subscribe/<id>` - 订阅内容详情（增强版）
- `GET /api/subscribe/stats` - 订阅统计数据
- `GET /api/subscribe/sources` - 订阅源列表
- `GET /api/subscribe/subscriptions` - 订阅源详情列表
- `GET /api/subscribe/sources-meta` - 订阅源元数据
- `GET /api/subscribe/digest` - 订阅摘要（智能排序）
- `GET /api/subscribe/source-weights` - 获取源权重
- `POST /api/subscribe/source-weights` - 设置源权重
- `GET /api/subscribe/by-source` - 按源分组内容
