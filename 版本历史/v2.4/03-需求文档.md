## 一、产品定位

Copaw新闻与Skills门户是一个集**新闻聚合**、**技能工具管理**、**订阅源管理**、**研发文档管理**于一体的综合平台。

### 核心功能模块
| 模块 | 功能 | 状态 |
|------|------|------|
| 新闻模块 | AI/科技热点新闻采集与展示 | ✅ |
| Skills模块 | AI Agent Skills工具展示 | ✅ |
| 订阅模块 | RSS/网站订阅源管理 | ✅ |
| 系统管理 | 版本化文档管理 | ✅ V2.4新增 |

## 七、API设计

### 7.1 新闻API
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | / | 首页 |
| GET | /news/{id} | 新闻详情 |

### 7.2 Skills API
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /skills | Skills列表 |
| GET | /skill/{id} | Skills详情 |

### 7.3 订阅API
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /subscribe | 订阅列表 |
| POST | /api/subscribe/add | 添加订阅 |
| POST | /api/subscribe/delete/{id} | 删除订阅 |

### 7.4 Wiki API
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /sys | 系统管理页 |
| GET | /api/wiki/list | 文档列表 |
| GET | /api/wiki/raw/{filename} | 获取文档内容 |
| GET | /api/wiki/meta/{filename} | 获取文档元数据 |
| POST | /api/wiki/{filename} | 保存文档 |
