## 五、部署说明

### 5.1 环境要求
- Python 3.12+
- Flask
- requests (测试用)

### 5.2 安装步骤
```bash
# 安装依赖
pip3 install flask requests --break-system-packages

# 启动服务
cd /home/zhang/.copaw/news_system
python3 app.py

# 运行测试
python3 test_v24.py
```

### 5.3 服务地址
- 新闻首页：http://localhost:5000
- Skills: http://localhost:5000/skills
- 订阅：http://localhost:5000/subscribe
- 系统管理：http://localhost:5000/sys
